This project leverages physiological data from the WESAD dataset to detect stress levels using machine learning. I processed signals from PPG and EDA sensors and extracted HRV features using NeuroKit2. I applied both time and frequency domain analysis and trained models like SVM and Random Forest to classify stress vs. baseline states.

The final model achieved strong performance with cross-validation and was visualized in an interactive dashboard for real-time analysis.
