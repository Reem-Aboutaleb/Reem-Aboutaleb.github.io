🧪 Neuro-Physio Dashboard

An interactive web app to visualize biosignals, detect stress, and explore PPG + EEG features — all in one place.

“I wanted to create a tool that could help researchers and clinicians visualize physiological data in a clean, interactive way. So I built one.”

🧠 What It Does

The Neuro-Physio Dashboard is a Streamlit-based application that displays and analyzes physiological signals like PPG and EEG. It includes:

  • 📊 Real-time signal plotting
  • 🔍 PPG peak detection
  • 📈 Stress classification using HRV & machine learning
  • 🧬 EEG activity simulation and visualization
  • 📂 Upload-your-own-signal & export reports

💡 Why I Built This

I wanted to bring together my knowledge of signal processing, biomedical engineering, and data visualization. By building a real app, I was able to:

  • 🛠 Practice full-stack development with Streamlit
