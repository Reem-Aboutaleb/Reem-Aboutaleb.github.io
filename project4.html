In this project, I simulated and visualized drug release kinetics for both zero-order and first-order release models using Python and Matplotlib. The simulation models the rate at which drugs are released from a coated microneedle over time, offering insights into therapeutic dosage control.

I developed clear plots to compare release curves between models and analyzed how parameters like release rate constants and time affect drug delivery performance. This tool can help in optimizing drug delivery systems for transdermal applications.
