I created this project to explore how we can use signal processing to extract features like R-R intervals, detect peaks, and calculate heart rate variability in Python.

üî¨ Project Pipeline

This project uses a combination of NumPy, Pandas, and Matplotlib to preprocess and analyze ECG signals.
